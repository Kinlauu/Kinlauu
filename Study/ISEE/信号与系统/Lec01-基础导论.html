    系统¶

    无记忆系统¶

    在给定时间的输出只依赖于同时的输入

    有记忆系统¶

    因果系统¶

    如果某一时刻的输出仅依赖于某一时刻的输入，则该系统是因果的，即系统不能预测输入。

    Same input => Same output

    √××√√

    Note

    只考虑x（t）内 t 是否会与未来相关（是不是输出在输入之后发生的），因果系统只依赖于当时刻，不能预测未来

    稳定系统-BIBO¶

    定义1: 稳定系统是指输入增量导致输出增量的系统。

    定义2: 如果每个有界输入都导致有界输出，系统是BIBO稳定的。我们将使用这个定义
