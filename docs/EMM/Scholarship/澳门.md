???+ note 
    奖学金/特别学科奖学金/贷学金 如果符合条件可以同时进行申请。
        但是需要注意的是，最后选择只能选取一个。

???+ tip
    如果都申请成功 个人觉得优先级为：指定学科奖学金＞奖学金＞贷学金   
    奖学金金额4230/月，指定学科5076/月
          奖学金有续期要求必须80分以上才能续期，相比之下指定学科的平均成绩合格更具有性价比。

???+ danger
    如果奖学金续期不上也没有关系，可以用成绩再做申请。
      但是换句话来说既然奖学金都续期不上，再申请通过的可能性并不高。所以大部分同学续期不上就直接转贷学金。

## 奖学金

| -------------- | --------------------- |

| - 内地         | 4,230.00             |

| - 台湾地区     | 4,230.00             |
| - 香港特别行政区 | 6,390.00             |
| -------------- | --------------------- |
| **其他国家或地区** | 6,390.00             |

### 偿还条件
- 奖学金无需偿还。

### 新申请条件
- **应届高中毕业生**：
  - 最近一学年的成绩为全科及格；
  - 最近一学年的平均成绩等于或超过16分（20分制）或80分（百分制），或在其学校名列第一或第二。
- **非应届高中毕业生**：
  - 最近一学年的成绩为全科及格；
  - 最近一学年的平均成绩等于或超过16分（20分制）或80分（百分制），或成绩达到“良”或以上程度。
- **博士生**：
  - 上一高等教育层次取得的平均成绩为合格；
  - 获得高等院校或其领导人员、教学人员及研究人员发出的推荐信。

### 续期条件
- **硕士及以下课程**：
  - 最近一学年的平均成绩等于或超过16分（20分制）或80分（百分制），且全科及格。
- **博士生**：
  - 最近一学年的平均成绩等于或超过13分（20分制）或65分（百分制），或取得导师对其研究的赞同意见。

## 贷学金

### 金额
- 贷学金金额根据申请人的家庭经济状况和课程类型确定，具体金额由教育基金行政管理委员会公布。

### 偿还条件
- 贷学金需分期偿还，每期还款额不得低于按公式计算的金额。
- 偿还期计算公式：`P = (n + 2) × 12`，其中`P`为偿还期，`n`为收取贷学金的年期。
- 如偿还金额低于或等于澳门元5万元，偿还期不得超过2年（24期）。
- 如偿还金额高于澳门元25万元但低于或等于澳门元40万元，偿还期延长2年（24期），总偿还期不得超过10年（120期）。
- 如偿还金额高于澳门元40万元，偿还期延长3年（36期），总偿还期不得超过11年（132期）。

### 新申请条件
- 申请人的家团每月总收入不超过下表的上限：

  | 家团成员人数 | 家团每月总收入上限（澳门元） |
  |--------------|------------------------------|
  | 1            | 17,400.00                    |
  | 2            | 31,960.00                    |
  | 3            | 44,080.00                    |
  | 4            | 53,560.00                    |
  | 5            | 60,480.00                    |
  | 6            | 67,400.00                    |
  | 7            | 74,320.00                    |
  | 8人或以上    | 81,080.00                    |

- 如申请人或家团成员在申请期紧接之前六个月内失业，相关人士的上一财政年度总收入不纳入家团总收入计算。

### 续期条件
- 贷学金的续期条件主要基于家庭经济状况的持续评估，申请人需提供最新的家庭收入证明和其他相关文件。
